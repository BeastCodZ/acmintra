# Statement Intelligence Engine - Frontend

A modern Next.js dashboard for visualizing financial transaction analysis.

## Features
- **Modern Dashboard:** Built with React 19, Next.js 16, and Tailwind CSS v4.
- **Real-time Analysis:** Instant upload and visualization of statement data.
- **Interactive Insights:** Plain-language health insights and AI-explained risk flags.

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Configuration
The frontend communicates with the backend at `http://127.0.0.1:8000`. Ensure the backend is running.
